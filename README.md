# Pharmacy/Healthcare Flask Web App

## Features
- Home, About, Contact, Gallery, Admin Panel
- Admin can upload photos/videos to Gallery
- SQLite database
- Green/mint-green healthcare UI
- Animated CSS medical background
- Responsive, secure, beginner-friendly, and well-commented

## How to Run
1. Install requirements: `pip install flask`
2. Run the app: `python app.py`
3. Visit http://127.0.0.1:5000/

## Admin Login
- Username: admin
- Password: password

## Folder Structure
- app.py: Main Flask app
- schema.sql: Database schema
- /static: CSS, JS, uploads
- /templates: HTML templates
- /instance: SQLite DB
